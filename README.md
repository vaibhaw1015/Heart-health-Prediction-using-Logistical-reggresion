# ❤️ Heart Health Prediction using Machine Learning

This is a web-based Heart Disease Prediction system built using:

- Python
- Flask
- Logistic Regression
- HTML + Bootstrap

## How it works
The user enters medical parameters and the model predicts whether the patient is at high or low risk of heart disease.

## How to Run

1. Clone repository  
2. Install requirements  
   pip install -r requirements.txt  
3. Run application  
   python app.py  
4. Open browser  
   http://127.0.0.1:5000
